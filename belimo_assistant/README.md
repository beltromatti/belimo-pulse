# Belimo Assistant

Belimo Assistant is a separate installer-facing prototype inside the Belimo Pulse repository.

It focuses on physical actuator testing and commissioning support rather than building-wide control. It combines:

- live telemetry from a real Belimo actuator
- diagnostic and commissioning workflows
- an MCP server exposing actuator tools
- an AI assistant for fault identification and configuration checks

The goal is to help installers and technicians validate setups, detect configuration mistakes, and diagnose actuator issues directly on hardware.
